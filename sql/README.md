# SQL for Data Engineering

This folder contains all SQL practice files from the Udemy course **Data Engineering for Beginners: Learn SQL, Python & Spark**.

## Course Sections Covered

| Folder | Course Sections | Topics |
|--------|----------------|--------|
| `basic_queries/` | Section 5 | SELECT, WHERE, JOINs, GROUP BY, ORDER BY |
| `aggregations_ranking/` | Section 6 | Window functions, cumulative aggregations, ranking |
| `troubleshooting/` | Section 7 | SQL debugging and error handling |
| `performance_tuning/` | Section 8 | Query optimization, indexes, execution plans |
| `exercises/` | Sections 9 & 10 | Practice exercises and solutions |

## Tools Used
- **PostgreSQL** – primary database for all SQL practice
- **pgAdmin / psql** – query execution and exploration

## How to Use
Each subfolder contains `.sql` files for that topic. Files are named descriptively (e.g., `window_functions.sql`, `inner_join_example.sql`).  
Run them against your local or cloud Postgres instance set up in Section 4.
