# Aggregations and Ranking in SQL

SQL practice files from **Section 6 – Cumulative Aggregations and Ranking** of the Udemy Data Engineering course.

## Topics Covered (Section 6)
- Window functions: OVER(), PARTITION BY, ORDER BY
- Ranking functions: ROW_NUMBER(), RANK(), DENSE_RANK(), NTILE()
- Cumulative aggregations: running totals, running averages
- LAG() and LEAD() for accessing prior and next row values
- FIRST_VALUE() and LAST_VALUE()

## Files in This Folder
| File | Description |
|------|-------------|
| `window_functions.sql` | OVER(), PARTITION BY examples |
| `ranking.sql` | ROW_NUMBER, RANK, DENSE_RANK, NTILE |
| `cumulative_aggregations.sql` | Running totals and averages |
| `lag_lead.sql` | LAG() and LEAD() functions |

## Why This Matters
Window functions are one of the most powerful SQL tools for data engineering — they allow you to compute aggregations without collapsing rows, making them essential for analytics pipelines.
