# PySpark DataFrame APIs

Python scripts using the PySpark DataFrame API from **Sections 33–38** of the Udemy Data Engineering course.

## Topics Covered

| Section | Topic |
|---------|-------|
| Section 33 | Getting Started with PySpark DataFrame APIs |
| Section 34 | Create Spark DataFrames using PySpark |
| Section 35 | Basic Transformations using PySpark DataFrame APIs |
| Section 36 | Joining Data using Spark DataFrame APIs |
| Section 37 | Ranking using PySpark DataFrame APIs |
| Section 38 | Integration of Spark SQL and PySpark DataFrame APIs |

## Files in This Folder
| File | Description |
|------|-------------|
| `create_dataframes.py` | Creating DataFrames from CSV, JSON, Parquet |
| `basic_transformations.py` | select(), filter(), withColumn(), alias() |
| `aggregations.py` | groupBy(), agg(), count(), sum() |
| `joins.py` | join() with inner, left, right, outer |
| `ranking.py` | Window functions: rank(), dense_rank(), row_number() |
| `sql_and_df_integration.py` | Mixing Spark SQL with DataFrame APIs |

## How to Run (Local)
```bash
pip install pyspark
python create_dataframes.py
```
