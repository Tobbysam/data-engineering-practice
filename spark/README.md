# Apache Spark (PySpark) for Data Engineering

This folder contains all Spark and PySpark practice files from the Udemy course **Data Engineering for Beginners: Learn SQL, Python & Spark**.

## Folder Structure

| Folder | Course Sections | Topics |
|--------|----------------|--------|
| `spark_sql/` | Sections 22-32 | Spark SQL queries, Delta Tables, window functions, aggregations, joins |
| `pyspark_df_apis/` | Sections 33-38 | PySpark DataFrame APIs, transformations, joins, ranking |
| `elt_pipelines/` | Section 39 | ELT Data Pipelines using Databricks |
| `performance_tuning/` | Sections 40-43 | Catalyst Optimizer, cluster config, columnar formats, partitioning |
| `hadoop_spark_cluster/` | Sections 44-53 | Hadoop HDFS, Hive, Dataproc, Spark applications |

## Tools and Platforms Used
- **Apache Spark / PySpark** - distributed data processing
- **Databricks** - managed Spark environment on Google Cloud Platform (GCP)
- **Google Cloud Platform (GCP)** - cloud provider for Databricks and Dataproc
- **Spark SQL** - SQL interface for structured data processing
- **Delta Tables** - ACID-compliant table format for data lakes

## How to Use
- spark_sql/ files are .sql files or Databricks notebooks (.ipynb or .py)
- pyspark_df_apis/ files are .py scripts using the PySpark DataFrame API
- Most examples require a Databricks or local Spark environment
