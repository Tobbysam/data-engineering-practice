## Key Concepts
- **Dataproc**: Google Cloud's managed Hadoop/Spark service — clusters provisioned on demand
- **HDFS**: Distributed file system — data stored in blocks across cluster nodes
- **spark-submit**: Command to submit PySpark jobs to a Spark cluster
