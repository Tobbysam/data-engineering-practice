# Python Basics for Data Engineering

Python fundamentals scripts from **Section 11 – Getting Started with Python** of the Udemy Data Engineering course.

## Topics Covered (Section 11)
- Python data types: int, float, str, bool, None
- Variables and assignment
- Conditional logic: if, elif, else
- Loops: for, while, list comprehensions
- Functions: defining, parameters, return values
- File I/O basics: reading and writing text files
- Modules and imports
