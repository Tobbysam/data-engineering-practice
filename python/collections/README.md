# Python Collections for Data Engineering

Scripts from **Section 12 – Python Collections for Data Engineering** of the Udemy Data Engineering course.

## Topics Covered (Section 12)
- Lists: creation, slicing, sorting, list comprehensions
- Tuples: immutable sequences, unpacking
- Dictionaries: key-value access, iteration, dict comprehensions
- Sets: deduplication, set operations (union, intersection, difference)
- Using collections to process structured data
- Functional patterns: map(), filter(), zip()

## Files in This Folder
| File | Description |
|------|-------------|
| `lists.py` | List operations and comprehensions |
| `dicts.py` | Dictionary usage for record-like data |
| `sets.py` | Set operations for deduplication |
| `tuples.py` | Tuples and unpacking patterns |
| `functional.py` | map(), filter(), zip() examples |

## Why This Matters for Data Engineering
Collections are the backbone of data transformation in Python — understanding how to efficiently manipulate lists of records and key-value data is foundational before moving to Pandas and Spark.
