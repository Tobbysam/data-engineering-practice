# Python for Data Engineering

This folder contains all Python practice files from the Udemy course **Data Engineering for Beginners: Learn SQL, Python & Spark**.

## Folder Structure

| Folder | Course Sections | Topics |
|--------|----------------|--------|
| `basics/` | Section 11 | Python fundamentals, data types, control flow, functions |
| `collections/` | Section 12 | Lists, dicts, sets, tuples for data processing |
| `pandas/` | Section 13 | DataFrames, filtering, joins, aggregations, file I/O |
| `troubleshooting/` | Section 16 | Debugging, error handling, common Python issues |
| `performance_tuning/` | Section 17 | Optimizing Python data pipelines |

> Projects are in the `/projects` folder at the root of this repo.

## Tools Used
- **Python 3.x**
- **Pandas** - data manipulation and analysis
- **psycopg2 / SQLAlchemy** - database connectivity (used in projects)

## How to Use
Each subfolder contains `.py` files named after the concept covered.
Run files locally using `python filename.py` or open in Jupyter/VS Code.
